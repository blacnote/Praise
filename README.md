# ChurchSense

**A worship-service operating system for the local church.**
*By DLGSENSE*

> **Core rule:** ChurchSense follows worship. Worship does not follow ChurchSense.

ChurchSense helps churches **plan, run, and control** praise-and-worship services — set building, separated stem playback, click + guide tracks (team ears only, never the house), audio routing, ProPresenter sync, and **Spirit Flow Mode** for Spirit-led worship that pauses, prays, loops, vamps, or extends a song beyond plan.

---

## This is a standalone DLGSENSE project
ChurchSense is its **own** product. It is **not** part of and must **not** be merged into:
PodSense · DreLovesGod-site · 1DROP · SingSense · HARMONY · ASCEND · SUMMIT · CLARITY · or any other app/repo.
No symlinks into other apps. No reusing another app's repo. See `docs/SAFETY_RULES.md`.
