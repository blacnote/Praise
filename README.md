# Praise

**A worship-service operating system for the local church.**
*By DLGSENSE*

> **Core rule:** Praise follows worship. Worship does not follow Praise.

Praise helps churches **plan, run, and control** praise-and-worship services — set building,
separated stem playback, click + guide tracks (team ears only, never the house), audio routing,
ProPresenter sync, and **Spirit Flow Mode** for Spirit-led worship that pauses, prays, loops,
vamps, or extends a song beyond plan.

---

## Standalone DLGSENSE project
Praise is its **own** independent project, sitting in `DLG Ideas/` alongside the others.
