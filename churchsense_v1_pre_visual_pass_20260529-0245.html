About this build (v1)

This is an honest proof-of-concept, built as a single self-contained HTML file (the same pattern as the DreLovesGod-site working files). It runs in any browser.

Working now

  • Set Builder: add songs, reorder, total time, jump to any song
  • Song data: title, artist, key, BPM, lyrics, sections + lengths
  • Click track (Web Audio metronome at BPM) + count-in
  • Guide cues: visual + spoken (browser speech)
  • Spirit Flow: Hold, Loop, Prayer/Pad, Resume — with live time recalculation
  • House mix kept clean of click/guide (enforced in logic)

Faked / not real yet (honest)

  • Stems are synthesized tones, not real separated audio files
  • True multi-output routing needs a desktop layer (browsers can't address multiple interfaces)
  • ProPresenter sync is a status placeholder, not a live link

Next

  • Load real multi-stem audio; make Spirit Flow loop/hold glitch-free with real stems (the critical bet)
  • Real ProPresenter network sync; plan the desktop layer for true routing
